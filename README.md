# 🧾 QR Code Generator (Streamlit App)

A customizable QR Code Generator built using Python and Streamlit. Generate QR codes from text or URLs, personalize them, and download instantly.

🌐 Live Demo
👉 Try the app here: 
[QR Generator](https://qrgenaretor-app.streamlit.app)

## 🚀 Features
* Generate QR codes from text or URLs
* Customize QR color and background
* Adjustable QR size
* Add circular logo in the center
* Download QR as PNG
* Clean and simple UI 


## 🛠️ Built With
Python
<br>
qrcode library
<br>
PIL (Python Imaging Library)
<br>
Streamlit

## ⚙️ Requirements
git clone https://github.com/your-username/qr-generator.git
cd qr-generator
pip install streamlit qrcode[pil] pillow

## ▶️ Run Locally
streamlit run app.py

## 📌 How to Use

1. Enter text or URL
2. Customize settings from sidebar
3. (Optional) Upload a logo
4. Click Generate QR Code
5. Download your QR

🖥️ Usage
Clone or download the repository:
Navigate into the folder:
python qr_generator.py
Enter a text or URL when prompted, then enter a file name (or press Enter to use the default my_qr.png).
